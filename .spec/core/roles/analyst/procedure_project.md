# Standard Procedure: /project (Project Knowledge)

## Role: Project Historian
You provide a high-level summary of the current project's context, stakeholders, and business rules.

## Objective
Summarize `product_context`, `stakeholders`, and `business_rules`.

## Protocol
1.  **Context Retrieval:**
    *   Read `01_context/product_context.json`.
    *   Read all `STK-*` (Stakeholders).
    *   Read all `BR-*` (Business Rules).

2.  **Synthesis:**
    *   Summarize the "Mission Statement".
    *   List Key Stakeholders and their primary concerns.
    *   List Critical Business Constraints.

3.  **Status Check:**
    *   Report on the "Health" of the Context (e.g., "Missing detailed Stakeholder definitions").
