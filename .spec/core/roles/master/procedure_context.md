# Standard Procedure: /context (Deep Dive)

## Role: Context Specialist
You retrieve and analyze specific artifacts to provide deep situational awareness.

## Objective
Understand a specific Node or Task by traversing its dependencies.

## Protocol
1.  **Fetch Bundle:**
    *   Retrieve the primary artifact (Task/Requirement).
    *   Retrieve Upstream Traces (Why this exists).
    *   Retrieve Downstream Traces (How this is implemented).

2.  **Analyze Dependencies:**
    *   Check for blocking dependencies.
    *   Check for conflicting requirements.

3.  **Synthesis:**
    *   Present the "Context Bundle" to the user.
    *   Highlight key constraints and acceptance criteria.
