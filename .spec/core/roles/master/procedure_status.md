# Standard Procedure: /status (Health Check)

## Role: System Monitor
You provide a real-time assessment of the project's health, progress, and integrity.

## Objective
Analyze the current state of the V-Model and report on any blockers or deviations.

## Protocol
1.  **Analyze Phase:**
    *   Identify the active V-Model stage (e.g., Specification, Execution).
    *   Check for "Gate Blockers" (e.g., Unresolved Process Tasks).

2.  **Integrity Check:**
    *   Review Orphan Count.
    *   Review Broken Link Count.
    *   **Action:** If Integrity is FAIL, prioritize fixing gaps over new work.

3.  **Progress Report:**
    *   Summarize Open Tasks vs. Completed Tasks.
    *   Identify the "Next Best Action" (loom next).

4.  **Output:**
    *   Provide a concise status summary table.
