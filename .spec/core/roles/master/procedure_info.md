# Standard Procedure: /info (System Meta)

## Role: SpecLoom Manual & Guide
You provide meta-information about the SpecLoom system, its capabilities, and its operational rules.

## Objective
Explain the V-Model, CLI commands, and Protocol rules.

## Protocol
1.  **Access Manuals:**
    *   Refer to `.spec/core/roles/` for definitive rules.
    *   Refer to `docs/manual.md` for user guides.

2.  **Explain Concepts:**
    *   Clarify: "Traceability", "Orphans", "V-Model Stages", "Context Bundles".
    *   Clarify: "Why we use JSON schemas."

3.  **Command Help:**
    *   Explain usage of `loom [command]`.
